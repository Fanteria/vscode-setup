# VS Code settings
This repository contains files for my setup of workspace in VScode. I'm using a few extensions (_all listed below_) of which is the most important Vim emulation for VS Code. Disclaimer: If you don't know what is vim, first check it before you experiment.

## Extensions

- [Markdown Preview Enhanced](https://marketplace.visualstudio.com/items?itemName=shd101wyy.markdown-preview-enhanced)
- [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)
- [REST Client](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)
- [Vim](https://marketplace.visualstudio.com/items?itemName=vscodevim.vim)
